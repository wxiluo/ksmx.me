--- categories: 榜样志, 感悟 date: 2008/11/18 17:21:21 guid: http://ufusion.org/?p=142 permalink: http://blog.ksmx.me/榜样志/advice-of-our-life tags: 人生, 忠告, 思考, 感悟 title: 一生的忠告 --- 这是人生的忠告，写下来，我们共勉： # 不睡懒觉。让每一天的早晨都充满活力。跑步、读英语是最适合在早晨做的两件事，它们将带给你一天的好心情和成就感！ # 跟别人说话时，一定要正视对方的眼睛。 # 对人多说一声“谢谢”，再进一步，请多说“非常感谢”。 Thanks. Thank you. Thank you very much. Thank you so much! # 对人多说一声“劳驾”，这不需要费任何力气，但却能产生美好的结果。 # 量入为出。 # 疯狂热爱阅读，阅读造就伟人。每天要找到一篇打动和提升你心灵的文章！ # 己所欲，施于人。 # 结交新朋友，珍惜旧朋友。 # 一定要严守秘密，你将会成为所有人的朋友。 # 学艺就老老实实去学，切勿浪费时间去学其中的“花招”。 # 承认错误。勇敢地承认错误。 # 要勇敢。要战胜恐惧。 # 永远不欺诈。 # 诚实永远是上策。 # 永远不要使别人失去希望。 # 学会聆听，机会抠门的时候是很轻的。 # 不要祈求上天赐予你身外之物，要祈求智慧和勇气。 # 愤怒时切勿行动。愤怒时，赶紧进行体育运动，或找个没人的地方大声发泄！在愤怒时采取行动，必将产生终身后悔的结果！ # 保持良好的姿态。踏进任何房间时，都要昂首阔步、充满自信。 # 不要在电梯里谈公事，你永远不知道站在旁边的人是否会偷听。 # 在完工之前，切勿付钱给人。 # 要赢得战争，不妨打一次败仗。 # 当心那些已没有什么可输的人。 # 执行艰巨任务时，要表现得只许胜、不许败。 # 切勿期望人生的公平。人生本来就是不公平的！ # 永远不要低估宽恕的力量。 # 遇事爱思考。 # 做事要大胆、勇敢。将来你回顾过去时，后悔没有做的事总会比后悔做了的事多。 # 改变世界的伟大主意往往是一个人独自埋头苦干得来的。 # 街头音乐家是个宝库。停下来听，然后给他一点钱。 # 不要拖拉，在必须做的时候做该做的事。 # 整洁条理。 # 不要怕说“我不知道”、“我不懂”。 # 不要怕说“对不起”、“是我的错”。 # 每日做笔记，对自己的一天进行总结。 # 写出你希望在死前做的25件事，把单子放在皮夹里，经常拿出来看看。你会成就很多你都不敢想象的成就。
